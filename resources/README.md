# Resources

Project runtime assets:

- `icons/app.ico`: Windows app icon
- `icons/app.icns`: macOS app icon
- `icons/app_icon.png`: source icon image
- `fonts/NotoSansCJKtc-Regular.otf`: bundled CJK font for matplotlib
- `presets/manifest.yaml`: whitelist for GUI built-in presets
- `presets/*.yaml`: packaged built-in presets shown in the GUI preset menu
